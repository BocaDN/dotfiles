# My Linux configuration files

#### Things to do after a fresh linux install:

##### add "username" to sudoers
```bash
EDITOR=vim visudo
```

```bash
%username_here   ALL=(ALL:ALL) ALL
```


#### Links to the tools and technologies I use with my Linux system:
- Linux distribution 1: [archlinux](https://www.archlinux.org/)
- Linux distribution 2: [debian](https://www.debian.org/)
- Text editor : [neovim](https://neovim.io/)
- Terminal multiplexer : [tmux](https://github.com/tmux/tmux/wiki)
- Unix shell : [z  shell](https://www.zsh.org/)
- Z shell framework : [oh-my-zsh](https://www.ohmyz.sh/)
- Enabling GUI on linux : [xorg](https://www.x.org/wiki/)
- File system : [btrfs](https://btrfs.readthedocs.io/en/latest/)
- File restoration utility (needs btrfs): [timeshift](https://github.com/linuxmint/timeshift)
- Window manager : [i3wm](https://i3wm.org/)
- Fuzzy finder : [fzf](https://github.com/junegunn/fzf)
